In Canada, a 40–48 hour instructor-led beginner Python course (with JS + Git + intro AI) can vary a lot depending on the provider type.

Here’s the realistic market range in 2026:

💰 1. Low-cost instructor-led courses (small academies / online schools)

  • $800 – $2,500 CAD total
  • Usually 30–50 hours live classes
  • Small groups or semi-live (Zoom + assignments + assistant support)
  • Limited career coaching

👉 Example: short “bootcamp-style” Python + basics programs

🎓 2. Mid-range professional bootcamps (most common)

  • $2,500 – $7,000 CAD
  • 6–10 weeks part-time OR intensive 1–2 week live bootcamp + homework
  • Includes: Live teacher, TA support, projects (GitHub, mini apps, basic AI intro)
  • Career guidance sometimes

👉 This is the most typical price range for what you described (40–48h structured + support)

🏫 3. University / branded bootcamps

  • $6,000 – $15,000+ CAD
  • Very polished, strong brand, structured curriculum
  • JS + Git + intro AI/data properly included
  • Networking + career services

📌 Reality check

A true 40–48 hour live instructor-led course with real support is usually not cheap like Udemy ($20–$100) because:

  • Instructor + assistant time is expensive
  • Live scheduling limits group size
  • Mentoring adds cost

Sweet spot: ~$2K–$5K CAD

⚡ Quick summary

  • Cheap live-ish course: $800–$2,500
  • Good instructor + TA bootcamp: $2,500–$7,000
  • Premium schools: $7,000–$15,000+
