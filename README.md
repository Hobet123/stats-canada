# stats-canada
Statistics Canada has developed a Web Data Service that provides access to a wide range of statistical data and metadata.

This website presents some of the most popular and interesting datasets through interactive charts and maps, 
making the data easier to explore and understand.

Historical data is also available, allowing you to analyze trends and compare changes over time.
